# Badminton Scoring Online

แอปพลิเคชันนับคะแนนแบดมินตันออนไลน์สไตล์ Apple Minimalist ออกแบบมาเพื่อการใช้งานที่ง่ายบนมือถือและคอมพิวเตอร์ ทำตามกติกาสากลของสหพันธ์แบดมินตันโลก (BWF)

## คุณสมบัติเด่น (Features)
- 🏸 **กติกาสากล (BWF Standard rules):** นับแบบ Best of 3 เซต, เซตละ 21 คะแนน, มีระบบดิวส์ (Deuce) เพื่อชนะด้วยคะแนนต่างกัน 2 คะแนน และลิมิตคะแนนสูงสุดที่ 30 คะแนน
- ⚡️ **ฝั่งเสิร์ฟลูก (Serving Indicator):** บอกฝั่งการเสิร์ฟอัตโนมัติ (คะแนนคู่ เสิร์ฟจากคอร์ทขวา / คะแนนคี่ เสิร์ฟจากคอร์ทซ้าย)
- 🔄 **สลับฝั่งสนาม (Switch Sides):** ปุ่มสลับฝั่งภาพผู้เล่นบนหน้าจอเพื่อช่วยให้ตรงกับสภาพการแข่งจริง
- 🔙 **กู้คืนคะแนน (Undo):** ปล่อยคะแนนผิดพลาดสามารถกู้คืน (Rollback) ย้อนกลับไปได้ไม่จำกัด
- 📊 **สถิติการเล่น (Progress):** วิเคราะห์สรุปผลหลังหรือระหว่างการแข่ง เช่น คะแนนรวมทั้งหมด, ช่วงที่ทำคะแนนได้ต่อเนื่องสูงสุด (Streak), และกราฟลำดับคะแนน
- 🏆 **ฉลองชัยชนะ (Celebration):** มีหน้าต่างการเฉลิมฉลองพร้อมเสียงสังเคราะห์ (Web Audio API) และเอฟเฟกต์กระดาษสี (Confetti) เมื่อการแข่งขันจบลง

## วิธีการใช้งาน
1. ดับเบิ้ลคลิก/กดที่ชื่อผู้เล่นเพื่อเปลี่ยนชื่อได้
2. กดคลิกที่การ์ดคะแนนของผู้เล่นเมื่อผู้เล่นคนนั้นได้คะแนน
3. คะแนนรวมแต่ละเซตจะถูกบันทึกที่ด้านล่างอัตโนมัติเมื่อสิ้นสุดแต่ละเซต
4. หากนับผิด ให้กดปุ่ม **Undo** เพื่อย้อนกลับคะแนน
5. กดปุ่ม **Progress** เพื่อดูสถิติและกราฟแสดงคะแนนย้อนหลัง
6. กดปุ่ม **Switch Sides** (ขวาบน) เพื่อสลับฝั่งของมุมมองการ์ดคะแนนผู้เล่นซ้าย-ขวา
